# Registro de Alumnos sin BD - App Android

# Características
- Nombre
- Edad
- Id de Alumno
- Visualización en una lista con diseño personalizado
- Actualización y eliminación de usuarios
- Diseño moderno con 'Material Design3'

# Funciones

- REGISTRAR: crea los datos insertados en los EditText sin que se muestren luego de insertarlos
- ACTUALIZAR: consulta y muestra los datos que se registraron
- ELIMINAR: eliminar todos los registros
- OPCIONES EXTRA: Al dar click a un registro te salta una ventana en donde puedes registrar la    salida y otra en eliminar solo ese registro.


# Tecnologías

- Java
- Android SDK
- SharedPreferences
- Gson
- Material Design
- ImageView + Galería para selección de imágenes

# Requisitos

- Android Studio
- Android API 21 o superior
- Permiso de acceso a galería (API >= 23)
